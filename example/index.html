Immersive Web Emulation Runtime Demo

  • No Native WebXR Support Detected: IWER has created an emulated Meta Quest 3 device and injected its WebXR runtime, you can now enter WebXR in emulation mode and control the emulated device via the GUI.
  • Action Playback Demo: This demo is loaded with a pre-recorded input session captured on a Meta Quest 3 device, you can view the playback by entering VR in emulation mode and click the "Play Action" button.
  • Native WebXR Support Detected: Enter VR and record your input session by pressing trigger on your right controller to start/end recording
